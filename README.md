## Featured Projects

### 1. mimi connect
スマートフォンの位置情報や音声入力をもとに、クラウド経由で家のロボットに指示を送り、行動させる生活支援アプリです。外出中のユーザーの状況に応じて、家のロボットが自律的に支援を行うことを目指しました。

- **担当**: バックエンド開発、ロボット側処理、iOS対応、位置情報取得機能
- **技術**: Flutter, Firebase, Cloud Text-to-Speech API
- **実績**: コンテスト2位入賞
- **Slides**: [Canvaリンク](#)

### 2. [ROS module for natural language robot commands](https://github.com/naminamina/command_understanding)
人の自然言語によるあいまいな指示を解釈し、ロボット制御用ミドルウェア ROS 上で実行可能な命令へ変換するモジュールです。  
「部屋を片付けて」「この人を案内して」などの指示を、ChatGPTを用いて行動レベルのコードに落とし込めるようにしました。

- **担当**: 自然言語解釈・ROS連携モジュール開発
- **技術**: ChatGPT, ROS

### 3. [John & Kosokoso](https://github.com/KIT-Happy-Robot/hackit_happy_robot)
スマートフォンアプリに投稿された文章からユーザーの感情を推定し、それに合わせてロボットが一緒に喜んだり悲しんだりする感情共有型サービスロボットです。

- **担当**: バックエンド開発、スマホアプリとロボットの接続処理、感情推定機能の実装
- **技術**: バックエンド開発, 感情推定, ロボット連携
- **Slides**: [Canvaリンク](#)

### 4. [4DX@HOME](https://github.com/jphacks/kz_2504)
AIによる動画解析と専用デバイスを組み合わせ、自宅で4DXのような映像体験を実現することを目指したプロダクトです。映像と同期して振動・光・風・水しぶきなどの演出を発生させ、臨場感のある体験を生み出します。

- **担当**: Reactを用いたフロントエンド開発、接続フロー設計、画面遷移設計、演出状況をリアルタイム表示するUI実装
- **技術**: React, UI/UX, device connection

### 5. [Chat_Mimi](https://github.com/naminamina/chat_mimi)
生活支援ロボット「Mimi」と会話できるボイスボットシステムです。ブラウザ上で会話すると、自作したLive2Dアバターが表情や動きをつけながら応答し、ロボットとのコミュニケーションに感情や親しみやすさを加えることを目指しました。

- **担当**: ボイスボット開発、アバター連携
- **技術**: Live2D, Illustrator, Web, Voice bot
